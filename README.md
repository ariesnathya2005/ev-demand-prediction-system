# EV Charging Demand Prediction System

## Features
- Real-time prediction
- Machine learning model (Random Forest)
- Streamlit dashboard with dynamic data visualizations
- Feature Importance and Model Performance tracking
- Cloud deployment using Render

## Tech Stack
- Python, Streamlit, Scikit-learn, Pandas, Matplotlib

Live App: https://ev-demand-prediction-system.onrender.com
